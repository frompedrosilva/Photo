<h1 align="center">Photo</h1>

<p align="center">
  <strong>A multilingual photography portfolio website designed to showcase creative work through an elegant and immersive visual experience.</strong>
</p>

<p align="center">
  Presenting photography collections, services, and professional information through a modern, responsive, and user-focused interface.
</p>

<hr>

<h2>Why Photo?</h2>

<p>
  Photography is about storytelling, emotion, and preserving moments. Photo was created as a digital portfolio platform focused on presenting visual work in a clean and engaging way.
</p>

<p>
  The project combines modern frontend techniques, responsive layouts, multilingual support, and interactive galleries to create a professional online presence for photographers.
</p>

<p>
  The goal is to provide visitors with an immersive browsing experience while making photography collections easy to explore across different devices.
</p>

<hr>

<h2>Features</h2>

<ul>
  <li>📸 Professional photography portfolio</li>
  <li>🌍 Multilingual support (English, Portuguese, Spanish)</li>

<h2>Installation</h2>

<p>
  Clone the repository:
</p>

<pre>
git clone https://github.com/frompedrosilva/photo.git
</pre>

<p>
  Navigate to the project directory:
</p>

<pre>
cd photo
</pre>

<p>
  Upload the project files to an Apache-compatible web server.
</p>

<p>
  Configure PHP handlers according to your hosting environment.
</p>

<p>
  Open:
</p>

<pre>
http://localhost
</pre>

<hr>

<h2>Configuration</h2>

<p>
  Contact form handlers are available for multiple languages:
</p>

<pre>
assets/www/handlers/

├── en/
├── es/
└── pt/
</pre>

<p>
  Update email settings according to your server configuration before deployment.
</p>
